# Changelog

All notable changes to this project will be documented in this file.

## [1.1.1] - 2026-04-20

### Fixed
- Simplified Smart Comments toggle logic for 100% reliable scrolling across all browser resolutions and sidebar layouts.

### Removed
- Legacy 'YouTubeGeminiSearch' options files (dead code cleanup).

## [1.1.0] - 2026-04-19

### Added
- Bi-directional video quality controls (Ctrl + Up / Ctrl + Down).
- Dynamic Video Quality HUD for immediate visual feedback.
- Interactive shortcut badges for direct navigation to browser settings.
- Stealth-focused UI overhaul with neutral grayscale palette and standardized typography.

### Fixed
- Native YouTube bug where shortcut menu failed to toggle closed in fullscreen mode.
- Alfabetical sorting of commands in browser settings via numeric prefixing.
- UI consistency across different screen resolutions and zoom levels.

### Removed
- Redundant 'Manage Keyboard Shortcuts' footer link.

## [1.0.0] - 2026-04-15

### Added
- Initial stable release of the YouTubeTweaks core.
- 'Comments on Right' sidebar layout optimization.
- Native YouTube AI (Gemini) sidebar toggle integration.
- Aspect ratio controls for fullscreen playback (2:1, 2.35:1, 4:3).
