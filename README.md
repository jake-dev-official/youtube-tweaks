# YouTubeTweaks: Power-Workflow Extension

A minimalist, high-performance browser extension to optimize your YouTube experience with native-feeling keyboard shortcuts and layout tweaks.

## 🚀 Features & Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| **Alt + ,** | **Ask AI** | Toggles the native YouTube Gemini "Ask" panel. |
| **Alt + .** | **Smart Comments** | Instantly scrolls to the comments or back to the video. |
| **Ctrl + ,** | **Sidebar Move** | Swaps the comments between the right sidebar and the bottom. |

## 🛠️ Local Installation

Since this extension is optimized for your personal workflow, you can install it manually in any Chromium-based browser (Chrome, Edge, Brave, Opera).

1. **Download the Code**: Download this repository as a ZIP and extract it to a folder on your computer.
2. **Open Extensions Page**:
   - Chrome/Brave: Go to `chrome://extensions`
   - Edge: Go to `edge://extensions`
3. **Enable Developer Mode**: Toggle the **"Developer mode"** switch in the top right corner.
4. **Load the Extension**:
   - Click the **"Load unpacked"** button.
   - Select the folder where you extracted the code.
5. **Pin it**: Click the puzzle icon in your browser toolbar and pin **YouTubeTweaks** for quick access to the toggle menu.

## 📦 Distribution Tips
To share this with others without giving them the folder:
1. Go to `chrome://extensions`.
2. Click **"Pack extension"**.
3. Select this project folder.
4. It will generate a `.crx` file. You can send this file to anyone—they just need to drag and drop it into their own Extensions page (with Developer Mode ON).
